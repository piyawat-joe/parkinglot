# parkinglot

#Project Requirements

Install JDK.
Install maven. (https://maven.apache.org/install.html)

#Build Instructions
Run the following command:
mvn clean install

How to run Parking Lot program

$   ./parking_lot

$   ./parking_lot file_inputs.txt
